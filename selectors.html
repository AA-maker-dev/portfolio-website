Selectors

1. Universal Selector

This selector applies styles to all elements.


2. Type Selector

This selector applies styles to all p elements.


3. Class Selector

This selector applies styles to certain tags with class


4. ID Selector

This selector applies styles to a specific element with an ID.


5. Pseudo-Class Selectors

This selector applies styles on the basis of user's interaction.

Hover over this link

6. Pseudo-Element Selectors

This selector applies styles to a specific part of an element.

7. Descendant Selector

Styles elements nested within a specified ancestor.

8. Child Selector

Styles elements that are direct children of a specified parent.

9. Adjacent Sibling Selectors

Styles elements that are immediately preceded by a specified sibling.

10. Group Selectors

Styles multiple elements with a single rule.
